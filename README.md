# Auri - Velas Artesanais Sustentáveis

Website para a marca Auri, especializada em velas artesanais naturais e sustentáveis.

## Sobre o Projeto

Site completo e responsivo que apresenta a marca Auri, seus valores, produtos e compromisso com a sustentabilidade. Desenvolvido com foco em design elegante e experiência do usuário.

## Características

- **Design Responsivo** - Funciona perfeitamente em desktop, tablet e mobile
- **100% HTML/CSS/JavaScript** - Sem dependências externas
- **Animações Suaves** - Efeitos de scroll e interações fluidas
- **Cursor Personalizado** - Efeito de glow que segue o cursor
- **Carrossel de Produtos** - Navegação interativa pelos produtos
- **Formulário de Contacto** - Sistema de contacto integrado

## Secções

- **Hero** - Apresentação da marca com visual impactante
- **Manifesto** - A essência e filosofia da Auri
- **Valores** - 4 pilares fundamentais (Natural, Sustentável, Feito à Mão, Produção Local)
- **Produtos** - Galeria de velas com diferentes aromas
- **Sustentabilidade** - Compromisso ambiental
- **Processo** - Como as velas são criadas
- **Contacto** - Formulário e informações de contacto

## Como Usar

1. Clone o repositório:
```bash
git clone https://github.com/mvttinif/velas-auri.git
```

2. Abra o arquivo `auri-website.html` no seu navegador

Ou simplesmente faça download do arquivo HTML e abra-o diretamente.

## Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (Vanilla)
- Google Fonts (DM Serif Display, Lora, Source Sans 3)
